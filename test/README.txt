IMPORTANT:
This program is developed with PHP, HTML, and JavaScript Language. Please use a PHP executable environment to run the program, such as Apache.

BEFORE YOU BEGIN:
Your computer will need to be connected to the Internet in order to execute the program.

USAGE:
1. Place the “group3” folder into your PHP executable environment.
2. Make sure the “group3” folder has read and write authority (0777).
3. Open a browser, and visit the URL of index.html in the “group3” folder. For example: http://localhost/group3/index.html
4. When the web page is loaded, click the button “click to generate XML file”.
5. The program will execute.
6. When the program finishes running, a “group3_result.xml” file will be generated in the “group3” folder (same path as index.html).
